## 🔨 Things I've Built

**[Nuclear Force: The Golden Hour Router](https://github.com/JoshuaViera/NuclearForce)**
A NYC-focused nuclear emergency survival app built during a fellowship hackathon. Static GeoJSON architecture, Leaflet/Turf.js stack, zero backend — because in an emergency, every millisecond counts.

**[MotionFrame](https://github.com/JoshuaViera/MotionFrame)**
Text-to-image-to-animated-video tool using HuggingFace's free API and browser-based Canvas/FFmpeg.wasm processing. AI creativity without the paywall.

**[Star Scavenger](https://github.com/JoshuaViera/StarScavenger)**
A full-featured browser-based space shooter in Next.js/TypeScript/Canvas — analytics, leaderboards, Supabase auth, mobile controls. Refactored from a 1400-line monolith into 10 modular hooks.

**[GitHub Learning Platform](https://github.com/JoshuaViera/GitHubLearningPlatform)**
Built for Pursuit students — Firebase auth, Firestore, a Git command simulator, D3.js branch visualization, and 24 interactive challenges. Deployed on Vercel.
